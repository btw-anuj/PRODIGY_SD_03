# Contact Manager

Simple single-page contact manager app (HTML/CSS/JS).

Features:
- Add, edit, delete contacts
- Search contacts
- Persistent storage using `localStorage`
- Export and import contacts as JSON

How to run:
1. Open [contact-manager/index.html](contact-manager/index.html) in your browser.
2. Or serve the folder and open `http://localhost:8000` (recommended to avoid file restrictions):

```bash
cd contact-manager
python3 -m http.server 8000
# then open http://localhost:8000
```

Files:
- [contact-manager/index.html](contact-manager/index.html)
- [contact-manager/styles.css](contact-manager/styles.css)
- [contact-manager/script.js](contact-manager/script.js)
